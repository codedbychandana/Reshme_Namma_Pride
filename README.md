# Purpose
**Reshme-Namma Pride** is an Android application designed to support sericulture farmers in 
monitoring and maintaining the ideal environmental conditions required for silkworm growth. 
Karnataka is known as the silk capital of India, and silkworms are highly sensitive to 
temperature and humidity changes. Even a small variation can damage or destroy the cocoon 
batch. Based on the silkworm growth stage, the app analyzes the data and provides smart 
suggestions and actionable alerts to help maintain a safe rearing environment.

## How to run
You may download this repository and use an integrated development environment such as Android Studio to build the apk file and install on any Android device (min. SDK 24).


## Scope and Features
The **scope** of the Reshme-Namma Pride project involves the development of a comprehensive 
Android-based digital supervisor tailored for the sericulture industry in Karnataka. 
The primary objective is to provide farmers with a robust tool to monitor and maintain sensitive 
environmental conditions, specifically temperature and humidity, which are critical to 
preventing the destruction of silkworm cocoon batches. By integrating stage-aware logic, the 
application transforms raw environmental data into actionable, "smart" advice based on the 
specific growth cycle or Instar stage of the silkworms. 

Technically, the project scope includes the implementation of a **Batch Tracker** to manage 
silkworm batches and a **Smart Advice Engine** to compare real-time readings against an ideal 
growth curve. 

To enhance data analysis, the application features **Automated Growth Progress 
Curves**, which generate visual graphs based on logged data. These curves provide a clear data 
visualization of temperature and humidity fluctuations over time, allowing farmers to spot 
trends and environmental inconsistencies at a glance. 

To ensure utility in rural settings, the 
scope encompasses an **Offline Functionality** using the Room Database, allowing for local 
record management without internet access. 

Additionally, the project includes a **CSV Export System** for generating shareable professional reports and **Kannada Language Support** to 
enhance accessibility for local farmers. 

The development was executed using a modern tech 
stack consisting of Kotlin, Jetpack Compose, and the MVVM architecture. While the current 
version focuses on manual logging and local data management, the identified future scope 
includes expanding the application's capabilities into Smart IoT Device Integration for 
automated climate control and AI-Based Predictive Analysis for disease risk assessment.
